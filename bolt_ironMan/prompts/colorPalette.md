# Kid Book Builder - Iron Man Inspired Color Palette

## Primary Colors
- **Hot Rod Red** `#E3000B` - Primary brand color, inspired by Iron Man's signature armor
- **Arc Reactor Blue** `#64F3FF` - For highlights and interactive elements, matching the arc reactor glow
- **Mark III Gold** `#FFB300` - Accent color for important UI elements and CTAs
- **Stark Industries Silver** `#C5C5C5` - For metallic UI elements and borders

## Secondary Colors
- **Armor Plate Gray** `#2A2A2A` - Dark backgrounds and text
- **Workshop Steel** `#4A4A4A` - Secondary text and UI elements
- **Repulsor White** `#FFFFFF` - Text on dark backgrounds and clean UI elements
- **Night Mode Black** `#121212` - Deep backgrounds and dark mode

## UI State Colors
- **Power-Up Green** `#00E676` - Success states and confirmations
- **Warning Orange** `#FF9100` - Warning states and notifications
- **Alert Red** `#FF1744` - Error states and critical alerts
- **Disabled Gray** `#757575` - Inactive elements

## Gradients
- **Arc Reactor Glow**: `linear-gradient(135deg, #64F3FF 0%, #0B6FFF 100%)`
- **Hot Rod Finish**: `linear-gradient(135deg, #E3000B 0%, #FF4B4B 100%)`
- **Gold Armor**: `linear-gradient(135deg, #FFB300 0%, #FF8F00 100%)`

## Usage Guidelines
1. Use Hot Rod Red as the primary brand color for main UI elements
2. Arc Reactor Blue should be used for interactive elements and highlights
3. Mark III Gold works well for CTAs and important UI elements
4. Stark Industries Silver for borders and subtle metallic accents
5. Use gradients sparingly for special elements that need to stand out
6. Maintain high contrast ratios for accessibility
7. Dark mode should primarily use Workshop Steel and Night Mode Black

## Opacity Variants
- High Emphasis: 100% opacity
- Medium Emphasis: 74% opacity
- Low Emphasis: 54% opacity
- Disabled: 38% opacity

## Accessibility Notes
- Ensure text maintains a minimum contrast ratio of 4.5:1
- Use Repulsor White text on dark backgrounds
- Armor Plate Gray text on light backgrounds
- Always test color combinations for accessibility compliance
