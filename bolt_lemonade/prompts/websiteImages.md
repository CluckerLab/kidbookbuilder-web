## 1. Hero Section Image
**Filename:** `01-hero-main.png`
**Prompt:** "Create a modern, split-screen digital illustration (1920x1080px, PNG format with transparency, 72 DPI). On the left: A child (age 8-10) sitting at a clean, modern desk, enthusiastically sketching in a notebook, surrounded by floating magical sparkles in Enchanted Purple (#7B2CBF) and Storybook Blue (#4361EE). On the right: A professional-looking online bookstore interface displaying a children's book cover with a small celebration notification showing '1st Sale!' in Dream Gold (#FFD700). The scene should be well-lit with Magic White (#FFFFFF) background and Soft Cloud (#F8F9FA) accents. Final image should be optimized for web (< 500KB). Style: Modern, clean, slightly magical but professional, suitable for a children's business platform."

## 2. Journey Icons Strip
**Filename:** `02-journey-icons.png`
**Prompt:** "Create a horizontal strip of four connected icons (1000x256px, PNG format with transparency, 72 DPI) showing the user journey. Include: 1) A glowing lightbulb with magical sparkles in Enchanted Purple (#7B2CBF), 2) A shining book with sparkle effect in Storybook Blue (#4361EE), 3) A rocket-powered book launching upward in Creative Coral (#FF6B6B), 4) A trophy with coins and stars in Imagination Green (#4CAF50). Connect icons with a flowing magical line in Dream Gold (#FFD700). Each icon should be clean, minimal, and slightly magical. Final image should be optimized for web (< 300KB). Style: Modern, cohesive icon set with consistent line weights and magical elements."

## 3. Features Illustration
**Filename:** `03-features.png`
**Prompt:** "Create a modern triptych illustration (1200x400px, PNG format with transparency, 72 DPI) showing three key platform features. Left panel: AI writing tools with floating text and magical elements in Enchanted Purple (#7B2CBF). Center panel: Kid-friendly business dashboard with charts and achievements in Storybook Blue (#4361EE) and Dream Gold (#FFD700). Right panel: Family safety features with shield and collaboration elements in Imagination Green (#4CAF50). Connect all panels with flowing magical elements. Final image should be optimized for web (< 400KB). Style: Clean, modern tech with magical accents, professional but child-friendly."

## 4. Age Group Journey
**Filename:** `04-age-journey.png`
**Prompt:** "Create a progressive illustration (1200x400px, PNG format with transparency, 72 DPI) showing three age groups (6-8, 9-11, 12-14) using the platform with increasing sophistication. Left: Young child with simple story elements and basic tools. Center: Pre-teen with more complex writing and first business steps. Right: Teen with professional tools and business achievements. Use full color palette progressively: Sparkle Pink (#FF69B4) for youngest, Storybook Blue (#4361EE) for middle, Enchanted Purple (#7B2CBF) for oldest. Connect scenes with flowing magical elements in Dream Gold (#FFD700). Final image should be optimized for web (< 400KB). Style: Modern, inclusive, showing clear progression in complexity while maintaining magical elements."
